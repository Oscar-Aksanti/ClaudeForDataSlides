# Jour 4 — Portfolio complet + stratégie carrière

## Formuler ses hypothèses avec Claude

*Chapitre 2*

```
Je travaille sur [ton cas choisi]. Ma question métier est :
[ta question].

Aide-moi à formuler 3 hypothèses testables avec les données disponibles,
et pour chacune, dis-moi quelle colonne du dataset permettrait de la vérifier.
```

## Lancer l'analyse de ton cas

*Chapitre 3*

```
Voici mon dataset nettoyé pour [ton cas].
Teste mes 3 hypothèses une par une, en précisant pour chacune
si elle est confirmée, infirmée, ou à nuancer — avec le chiffre à l'appui.
```

## Formuler une recommandation actionnable

*Chapitre 3*

```
À partir de mes résultats, rédige une recommandation unique,
formulée comme une décision que quelqu'un pourrait prendre dès demain.
Pas de généralité — une action précise, avec le chiffre qui la justifie.
```

## Construire une page portfolio avec Claude

*Chapitre 4*

```
Construis une page portfolio simple présentant mon projet [ton cas] :
contexte, méthode, résultat, recommandation. Design sobre,
une seule couleur d'accent, optimisé mobile.
```

## Rédiger un message de prospection

*Chapitre 5*

```
Aide-moi à écrire un message court pour [organisation ciblée],
qui mentionne un problème précis que j'ai remarqué et propose
une preuve concrète (mon projet [ton cas capstone]) plutôt qu'une candidature générique.
```
