# Maquettes de dashboards — lire avant usage

Ces deux images sont des **maquettes cibles** (wireframes stylés aux couleurs de la formation),
pas de vraies captures d'écran Power BI ou Excel. Elles montrent la structure et le niveau de
finition à viser — KPI en tête, visuel principal, tableau des lignes à vérifier en dessous —
pas le rendu exact du logiciel que tu vas utiliser.

- **Maquette-Dashboard-Jour1-CentreSante.png** — cible pour le dashboard de suivi de stock du Jour 1.
- **Maquette-Dashboard-Jour2-ONG.png** — cible pour le dashboard Power BI du programme ONG du Jour 2.

Pendant la formation, la vraie capture d'écran de ton dashboard — construit en direct dans Excel
ou Power BI — remplace cette maquette. C'est volontaire : la recherche en pédagogie montre qu'une
vraie capture de l'outil que l'apprenant utilise vaut largement mieux qu'une image générique, même
bien faite.
